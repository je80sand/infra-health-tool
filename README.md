# Infrastructure Health Tool

A simple, human-readable Python CLI tool that performs a basic infrastructure health check on a system.

This project collects system metrics, scans logs for common issues, and generates structured reports for troubleshooting and monitoring.

---

## What This Tool Does

- Collects system health metrics:
  - CPU usage
  - Memory usage
  - Disk usage
  - Operating system info
- Scans log files for common problem keywords
- Prints a clear summary to the terminal
- Saves a timestamped JSON report for later review
